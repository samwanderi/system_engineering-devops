# 0x01-shell_permissions

This repository contains Bash scripts that help you learn and practice Linux shell permissions. Each script corresponds to a specific task in handling files, directories, and permissions in a Linux environment.

## Scripts Included
1. `0-iam_betty` – Switches the current user to `betty`.
2. `1-who_am_i` – Prints the effective username of the current user.
3. `2-groups` – Displays all groups the current user belongs to.
4. `3-new_owner` – Changes the owner of a file named `hello` to `betty`.
5. `4-empty_file` – Creates an empty file named `hello`.
6. `5-execute` – Adds execute permission to the owner of `hello`.
7. `6-multiple_permissions` – Sets specific permissions on `hello`.
8. `7-complex_permissions` – Sets mode `-rwxr-x-wx` on `hello`.
9. `8-James_Bond` – Sets unusual permission patterns.
10. `11-directories_permissions` – Adds execute permission to subdirectories.
11. `12-directory_permissions` – Creates a directory with mode `751`.
12. `13-change_group` – Changes the group owner of `hello` to `school`.
